Introdução à Linguagem R

R é uma linguagem de programação e um ambiente de software livre para computação estatística e gráficos.

Conceitos Básicos

  • Vetores: Um dos tipos de dados mais básicos em R.
  • Data Frames: Estruturas bidimensionais que permitem armazenar diferentes tipos de dados.
  • Funções: Blocos de código que podem ser reutilizados.
